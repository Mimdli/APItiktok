## 接口填写示例

```
### [接口名称]
- **URL**: `GET /api/example`
- **功能描述**: 简要说明接口作用
- **请求参数**:
  | 参数名 | 类型 | 必填 | 说明 |
  |:---|:---|:---:|:---|
  | id | Long | 是 | 用户ID |
- **请求体** (JSON):
  ```json
  {
    "field": "value"
  }
  ```
- **响应示例** (JSON):
  ```json
  {
    "code": 200,
    "data": {},
    "msg": "success"
  }
  ```
```
